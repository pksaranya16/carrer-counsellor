# 🎓 NextGen AI Career Ecosystem

## 📌 Project Overview

NextGen AI Career Ecosystem is an intelligent career guidance platform developed using **Python, Natural Language Processing (NLP), and Streamlit**. The application helps students and job seekers discover suitable career paths based on their interests, skills, and personality traits.

The platform combines two approaches:

* **AI Text Predictor:** Uses NLP techniques to analyze user descriptions and recommend careers.
* **Psychometric Assessment:** Uses aptitude and interest-based questions to identify the most suitable career domain.

---

## 🚀 Features

### 🔮 AI Text Predictor

* Natural Language Processing (NLP) based career prediction.
* Tokenization and stop-word removal using NLTK.
* Keyword-based career mapping.
* Provides career recommendations and next-step guidance.

### 🧠 Psychometric Assessment

* Personality and aptitude-based assessment.
* Interactive slider-based questionnaire.
* Displays career alignment scores.
* Provides career recommendations based on interests.

### 📊 User-Friendly Interface

* Interactive web application built using Streamlit.
* Real-time predictions and visual feedback.
* Clean and responsive dashboard.

## 🧠 How It Works

### AI Text Predictor Workflow

1. User enters skills, hobbies, or interests.
2. Text is converted into tokens.
3. Stop words are removed.
4. Keywords are matched against the career database.
5. Best career recommendation is displayed.

---

### Psychometric Assessment Workflow

1. User answers aptitude and interest questions.
2. Scores are calculated for each career domain.
3. The highest score determines the recommended career path.
4. Career alignment metrics are displayed.
